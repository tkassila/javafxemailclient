Gmail email counter application

What it will do?

When a user is given his/her username or email address and password for a gmail email account, the application makes gmail email server connect and calculates how many email messages on different email addresses there are. And are listing result in the list constrol.

Purpose of the application

The result are listing at first those emails, which have most messages. By that is good to know which email messages is best to search on the browser and possible to delete from a gmail email server.

Features

  • When all messages are calculated, you can search some text from messages (from start, next and previous).
  • When you mouse clicked on a result email row, that email will be copyed into the clopboard of (Windows) OS.
  • You can cancel the email execution by pressing a button, when it is active.
  • And continue later by loading old result from the data file (a button) and pressing then a execution button.
  • You can show given password characters visually by pressing a checkbox.

Commmandline user parameters

java app .... -gui|-commandline -use_user_home|-donotuse_user_home [username password]

See inside of javaemailclient.bat or javaemailclient.sh

-gui = javafx gui is starting.

-commandline = a commandline app is starting.

use_user_home = a data file will be saved into a directory of HOME or like environment (=user home).

donotuse_user_home = a data file will be saved into a directory of a application installelation directory. The datafile name is: emailcount.txt
